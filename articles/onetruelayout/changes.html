Change Log

25 April 2006

Added

  • New appendix page, Problems with the Equal Height Columns method, explaining what problems have been found, suggesting some ideas to counter them and providing a bit of background. Of course, this should actually have been added back in mid-November for which I can only apologise.
  • Brief notes on the conclusion page and in the results tables as to how the methods and examples hold up in IE7 beta 2 (just fine it would seem).
  • Support for direction:rtl (hat tip to Rob Cochrane).

Fixed

  • Moved #footer { clear: both; } to the main inline CSS since that's where it should have been all along.
  • Validation errors on the Tucows example

3 November 2005

Added

  • News about the fix for Gecko 1.8 / Firefox 1.5.
  • Comments on the limitations of the percentage-based examples.
  • Ids for all the headings.
  • Remarks for people who know little about humour and sarcasm.

Fixed

  • Basic Any Order example now actually works in IE6 (reported by Andrey Petrov).
  • Text select scolling problem in Firefox 1.0 on the Nested Rounded Corners example. The same issue on the BBC homepage example has not been fixed because I am lazy.
  • Validation issues with the article pages themselves.

24 October 2005

Added

  • Breaking news about Opera 9 beta.
  • Two "real world" makeover examples - BBC homepage and Tucows fragment.
  • Results for iCab 3.

Fixed

  • All examples relying on the vertical grid method updated to work in both Opera 8 and 9.
  • Example pages did not have Safari hack applied to the CSS and HTML output blocks, rendering them unselectable/unclickable.
