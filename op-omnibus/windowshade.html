The Opera 6 Windowshade bug

This page demonstrates the Opera Windowshade Bug. It occurs with Opera 6 (not Opera 7), and is caused by the following combination:

A fixed div
A block with position:fixed and overflow:hidden
An interior element
Something inside the fixed div which goes beyond the bounds of its parent. This could be an image that is wider than the fixed div. This example uses a div with a negative margin.
Some scrolling content
The bug occurs when the page is scrolled, so the normal content of the page must be sufficiently long.

To see the effect, scroll down: the bottom border of the pink block zips up like a window shade!

Note also that Opera does not correctly display the border on the fixed div; also because of the overflow:hidden property.
